# memo-polycanva.complex

BlockNote を使った Notion ライクなメモツールです。

## 機能

- BlockNote リッチテキスト編集
- コードブロックの構文ハイライト
- サイドバーでページ管理（階層・選択・追加・ドラッグ並べ替え）
- 子ページの折りたたみ / 全展開・全折りたたみ
- 通常表示 / ごみ箱で複数選択して一括操作（削除・復元・完全削除）
- ページ検索（タイトル + 本文）
- ピン留めページの上位表示
- ごみ箱（論理削除 / 復元 / 完全削除）
- 最終更新日時の表示
- 場所別のコンテキストメニュー（ページ / サイドバー空白 / エディタ）
- コマンドパレット（`/` と `@` 接頭辞対応）
- ショートカット・コマンドのヘルプウィンドウ
- IndexedDB への自動保存
- JSON エクスポート / インポート（旧形式データ互換）

## ショートカット

- `Ctrl/Cmd + K`: 検索入力へフォーカス
- `Ctrl/Cmd + Shift + N`: ルートページを新規作成
- `Ctrl/Cmd + N`: 現在のページに子ページを作成
- `Ctrl/Cmd + R`: 現在のページ名を変更
- `Ctrl/Cmd + Shift + P`: 現在のページのピン留め切替
- `Ctrl/Cmd + Delete`: 現在のページをごみ箱へ移動
- `Ctrl/Cmd + P`: コマンドパレットを開く
- `Ctrl/Cmd + /`: ヘルプウィンドウを開く
- `/` または `@`: コマンドパレットを接頭辞付きで開く

## Google同期セットアップ（GAS + スプレッドシート）

このアプリはローカル保存（IndexedDB）を維持したまま、任意でGoogle同期を有効化できます。  
同期は「ワークスペース全体JSON」を 1 同期キーにつき 1 レコードとしてスプレッドシートへ保存します。

### 1. スプレッドシート作成

1. Google スプレッドシートを新規作成
2. URL または ID を控える

### 2. GAS貼付

1. [Google Apps Script](https://script.google.com/) で新規プロジェクト作成
2. リポジトリの `google-sync.gs` の内容を貼り付けて保存

### 3. Web公開

1. 「デプロイ」→「新しいデプロイ」
2. 種類は「ウェブアプリ」を選択
3. 実行ユーザーは自分、アクセス権は利用要件に応じて設定（一般には「全員」）
4. 発行された Web アプリ URL（`.../exec`）を控える

### 4. アプリにURL設定

アプリのサイドバー「同期設定」で以下を入力:

- GAS WebアプリURL
- スプレッドシートURLまたはID
- 同期キー（全端末で同じ値）
- 端末名（端末ごとに別値推奨）

### 5. 接続確認

1. 「接続テスト」を実行
2. 成功後、「今すぐ同期」でデータを送受信
3. 必要に応じて「クラウドから復元」を実行

## ユーザー間共有機能

GAS + スプレッドシートを使い、**異なるユーザー間でワークスペースのスナップショットを共有**できます。  
共有は読み取り専用のコピーとして相手側にインポートされ、互いのワークスペースは独立したまま維持されます。

### 共有の流れ

1. **発行者**（共有元）がサイドバーの「共有機能」→「このワークスペースを共有する」を押して共有キーを発行する
2. 発行者が表示された **GAS URL** / **スプレッドシートURL** / **共有キー** の 3 点を受信者に渡す
3. **受信者**がサイドバーの「共有機能」→「共有からインポート...」を開き、渡された 3 点を入力してインポートする

### 共有を発行する

1. Google 同期設定（GAS URL・スプレッドシート）を先に完了してください
2. サイドバー下部の「共有機能」アコーディオンを開く
3. 「このワークスペースを共有する」を押す
4. 発行後に表示される **共有キー・GAS URL・スプレッドシートURL** をコピーして受信者に渡す
5. ワークスペースを更新したあとに「共有を更新（再公開）」を押すと最新状態で上書き保存される
6. 「共有を取り消す」を押すとスプレッドシートから削除される（受信者の手元のコピーには影響しない）

### 共有からインポートする

1. サイドバーの「共有機能」→「共有からインポート...」を押す
2. モーダルに以下を入力する
   - **発行者のGAS WebアプリURL**
   - **発行者のスプレッドシートURLまたはID**
   - **共有キー**
3. 「インポート」を押す
4. ダイアログで以下を選択する
   - **OK（マージ）**: 現在のワークスペースにページを追加する
   - **キャンセル（上書き）**: 現在のワークスペースを共有データで置き換える
5. インポート後は「最新を再取得」ボタンで同じ共有の最新版をいつでも取得できる

### セキュリティ上の注意

- 共有キーは UUID（128bit）なので推測困難ですが、**GAS URL + スプレッドシートURL + 共有キーの 3 点を知っている人は誰でも取得可能**です
- 機密データを共有する場合は GAS のアクセス権設定（実行ユーザーの制限など）を適切に行ってください
- 共有スナップショットはスプレッドシートの `share_records` シートに保存されます。不要になったら「共有を取り消す」で削除できます

## よくある失敗と対処

- WebアプリURLが `.../exec` でない  
  - デプロイ済み Web アプリの URL を再確認してください
- スプレッドシートURL/IDが誤っている  
  - `/spreadsheets/d/<ID>` の `<ID>` 部分、または正しいURLを入力してください
- GAS側の公開範囲不足でアクセス不可  
  - Web アプリのアクセス権設定を見直して再デプロイしてください
- 権限エラー（SpreadsheetApp open 失敗）  
  - GAS 実行ユーザーに対象スプレッドシートへの権限があるか確認してください
- 同期キー不一致で端末間データが共有されない  
  - 複数端末で同じ同期キーを設定してください
